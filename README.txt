Чертоги фрилансера версия 3.0
---------------------------------------------------------------------------

1. Надо сделать мультиязычность сайта. Есть только английская версия.
Когда кто то заходит на сайт должна запускатся версия сайта, которая соответвует пользователю,
если в браузере посетителя сайта язык стоит ru - тогда должен запускатся сайт например www.test-lang.com/ru/
если в браузере посетителя сайта язык стоит tr - тогда должен запускатся сайт например www.test-lang.com/tr/ и т.д.
если совпадений не найдено, тогда www.test-lang.com   (это язык по умолчанию, английский)

Еще должна быть возможность в будущем добавить еще язык.

Некоторые вещи можешь делать по своему из своего личного опыта, но сделать надо используя именно эту сборку.

JavaScript находится в директории src/js/files/script.js

Какие языки будут:
1. English (по умолчанию)
2. Русский 
3. Казахский 
4. Турецкий 

У каждого языка будет своя отдельная HTML страница, но CSS стиль будет один.
